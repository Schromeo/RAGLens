# SledTrace v0.4.1 - Rebrand

## Release Title

SledTrace v0.4.1 - Rebrand

## Tagline

Local-first observability and debugging for RAG pipelines.

## Summary

SledTrace v0.4.1 is a compatibility-preserving rebrand release for the project previously named RAGLens.

This release keeps core behavior unchanged:

- warning logic remains deterministic-first
- collector and dashboard API semantics are unchanged
- SQLite schema and table names are unchanged
- existing local data remains accessible

## Compatibility Highlights

- New preferred collector env var: `SLEDTRACE_COLLECTOR_URL`
- Legacy env var still supported (deprecated): `RAGLENS_COLLECTOR_URL`
- New primary launcher: `scripts/start-sledtrace.py`
- Legacy launcher still supported via wrapper: `scripts/start-raglens.py`
- New preferred Python import path: `sledtrace`
- Legacy Python import path remains supported: `raglens`

## Notes

- v0.4.0 was originally released under the RAGLens name
- This release prepares migration paths without forcing immediate breaking changes
